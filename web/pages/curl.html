Replace URS_USER with your Earthdata Login username and URS_PASSWORD with your Earthdata Login password. If this file does not exist, create a new one. Ensure that you are the only one that can read this file, then follow the steps below.

NOTE: If these files require authentication from Earthdata Login for downloading over HTTPS, you may need to update the command above to what is described on this page - How to Access Data with cURL and Wget.


Mac OS X / Linux

1) Copy the Download Commands above and paste into a text document. Save it as "download.sh"
2) Execute the following command to download all of your requested files: 

sh ./download.sh

Windows

1) Copy the Download Commands above and paste into a text document. Save it as "download.bat"
2) Execute the following command to download all of your requested files: 

download.bat
