Using wget to Bulk Download Your Data

Since some of your download links are password-protected, you will need to add the netrc entries:

to the following file:

  • Windows: _netrc file found in your home folder.
  • Mac or Linux: .netrc file found in your home directory.

Replace URS_USER with your Earthdata Login username and URS_PASSWORD with your Earthdata Login password. If this file does not exist, create a new one. Ensure that you are the only one that can read this file.

1) Copy the links above and paste into a text document. Save it as "links.txt"
2) Execute the following command to download all of your requested files: 

wget --input-file=links.txt

Learn more about How to Access Data with cURL and Wget over HTTP from a web server configured for Earthdata Login authentication.
